\section{Ejemplos para secciones}
Lo abordado a partir de este punto es solo un ejemplo de cómo se pueden escribir fórmulas, teoremas, definiciones y gráficos en el documento. \\

Para definir un vector en $\mathbb{R}^3$:
\begin{equation}
    \vec{v} = \langle x, y, z \rangle = x\mathbf{i} + y\mathbf{j} + z\mathbf{k}
\end{equation}

\begin{mydefinition}{Producto Punto}
    El producto escalar de dos vectores $\vec{u}$ y $\vec{v}$ se define como:
    \[ \vec{u} \cdot \vec{v} = |\vec{u}| |\vec{v}| \cos(\theta) \]
\end{mydefinition}

\section{Derivadas Parciales}

El gradiente es fundamental en este curso. Utilizando el paquete \texttt{physics}, podemos escribirlo fácilmente:
\[ \grad f = \left( \pdv{f}{x}, \pdv{f}{y}, \pdv{f}{z} \right) \]

\section{Visualización de Superficies}



A continuación, un ejemplo de cómo graficar una superficie básica (paraboloide) directamente en tu documento:

\begin{center}
\begin{tikzpicture}
\begin{axis}[
    view={60}{30},
    axis lines=center,
    xlabel=$x$, ylabel=$y$, zlabel=$z$,
    colormap/viridis
]
\addplot3[
    surf,
    domain=-2:2,
    domain y=-2:2,
    samples=30
] {x^2 + y^2};
\end{axis}
\end{tikzpicture}
\end{center}

\begin{mytheorem}{Regla de la Cadena}
    Si $z = f(x, y)$ es diferenciable y $x=g(t), y=h(t)$, entonces:
    \[ \frac{dz}{dt} = \pdv{f}{x}\frac{dx}{dt} + \pdv{f}{y}\frac{dy}{dt} \]
\end{mytheorem}
